Moderator CP Permissions (1.0.3) for MyBB 1.6
Created by: Starpaul20
Copyright: ©2011
License: GPL

This plugin allows you to limit what sections of the Moderator CP a specific usergroup can use (edit profile, ban user, etc.). Please note permissions can only be set for Moderators, Administrators can always use the Moderator CP.

The usergroup permissions are located under the 'Moderator CP' tab when editing a usergroup. This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Install & Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you only need to overwrite old files with the new ones, no need to deactivate.
